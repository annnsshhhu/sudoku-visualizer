# 🧩 Interactive Sudoku Solver & Visualizer

An interactive web-based Sudoku solver with real-time backtracking visualization.

## 🚀 Live Demo
👉 https://annnsshhhu.github.io/sudoku-visualizer/

## ✨ Features
- Solve any custom Sudoku input
- Step-by-step backtracking visualization
- Speed control (slow → instant)
- Highlighting of row, column, and subgrid
- Stop / Reset functionality

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Async/Await)

## 🧠 Algorithm
- Backtracking with constraint validation

## ▶️ Run Locally
1. Download the project
2. Open `index.html` in browser
